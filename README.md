## 🚀 Proyek & Riset Saya

Berikut adalah beberapa sistem dan proyek riset yang telah saya bangun dan kembangkan:

*Optimasi Object Detection pada Arsitektur YOLOv4*

- **Tech Stack:** Python, OpenCV, Darknet / PyTorch, Roboflow.
